Main Processor:

The 'Brain' of Wall-e is provided by a Raspberry Pi Zero W board

The programming language used is a Python 3 script using a Flask Framework for the API

Needed to use Apache2 as the webserver to be able to use WSGI. Tried lighttpd but was unable to get lip WSGI to work.

Added 'www-data' user to 'sudoers' file and also 'gpio' Group to prevent errors with permissions executing the python script as root access is required for GPIO functions.
